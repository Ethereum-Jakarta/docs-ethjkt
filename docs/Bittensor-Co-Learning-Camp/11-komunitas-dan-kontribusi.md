### 1. Konten & Edukasi

| Kontribusi | Contoh | Dampak |

| Kontribusi | Contoh | Skill |

| **Documentation** | Perbaiki/lengkapi docs resmi | Technical writing |

### 3. Community Building

| Kontribusi | Contoh | Impact |

## üìã Cara Mulai Berkontribusi

### Step 1: Join Komunitas

```
1. Join Discord Bittensor ‚Üí perkenalkan diri
2. Join HackQuest Indonesia ‚Üí ikuti co-learning camp
3. Join Ethereum Jakarta ‚Üí networking lokal
```

### Step 2: Pahami Ekosistem

```
1. Baca dokumentasi resmi (docs.bittensor.com)
2. Selesaikan co-learning camp ini
3. Eksplorasi Taostats untuk memahami data jaringan
4. Ikuti diskusi di Discord
```

### Step 3: Mulai Kontribusi Kecil

```
1. Perbaiki typo di dokumentasi (GitHub PR)
2. Jawab pertanyaan newcomer di Discord
3. Tulis thread singkat tentang apa yang Anda pelajari
4. Test subnet di testnet dan laporkan bug
```

### Step 4: Level Up

```
1. Submit substantial code contribution
2. Buat tool/dashboard untuk komunitas
3. Adakan workshop/meetup
4. Ikuti hackathon/ideathon
```

## üéØ Langkah Selanjutnya

Setelah menyelesaikan co-learning camp ini, ada 3 langkah konkret yang bisa Anda ambil:

### Action Item 1: Setup dan Jalankan Miner

**Timeline:** 1-2 minggu

```
‚ñ° Install Bittensor dan btcli
‚ñ° Buat wallet (coldkey + hotkey)
‚ñ° Pilih subnet yang sesuai skill Anda
‚ñ° Jalankan miner di testnet
‚ñ° Benchmark performa
‚ñ° Jika profitable ‚Üí migrasi ke mainnet
```

### Action Item 2: Bangun Sesuatu untuk Ekosistem

**Timeline:** 2-4 minggu

```
‚ñ° Identifikasi kebutuhan yang belum terpenuhi
‚ñ° Desain solusi (tool, dashboard, library)
‚ñ° Build MVP
‚ñ° Share ke komunitas untuk feedback
‚ñ° Iterasi berdasarkan feedback
```

### Action Item 3: Ikuti Bittensor x HackQuest Ideathon

**Timeline:** Sebelum 28 Feb 2026

```
‚ñ° Baca detail ideathon di modul selanjutnya
‚ñ° Bentuk tim (opsional)
‚ñ° Brainstorm ide subnet
‚ñ° Buat proposal dengan template yang disediakan
‚ñ° Submit sebelum deadline
```
